# Ọkwa Sandboxing (Sandboxing Layers)

OmniPaw na-eji **Nkewapụ Na-aga n'Ihu (Progressive Isolation)**. Ọ bụghị ọ bụla ngwa ọrụ chọrọ Docker container zuru ezu, mana ọ bụla ngwa ọrụ ga-abụ nke a dịpụrụ adịpụ. Ihe nchịkwa na-akwado ọkwa atọ dị iche iche nke sandboxing dabere na ihe ize ndụ nke ngwa ọrụ ahụ.

## L1: Nkewapụ Process Ọkọlọtọ

A na-eji ya maka **Ngwa Ọrụ Logic (Pure Logic Tools)** (dịka nchịkọta unit, string formatters). Ngwa ọrụ ndị a na-agba n'usoro Node.js child process dịpụrụ adịpụ mana ha na-ekerịta faịlụ sistemụ nnabata (ọ bụ ezie na ọ bụ naanị maka ọgụgụ).

- **Uru**: Ọsọ dị elu.
- **Ihe Ize Ndụ**: Enwere ike ịgbanahụ site na nkwụsị Node.js native vulnerabilities.

## L2: Nkewapụ Chroot / Jail

A na-eji ya maka **Ngwa Ọrụ Faịlụ Sistemụ** (dịka ịgụ ndekọ, ide faịlụ nwa oge). Ihe nchịkwa na-emepụta "root" directory mebere emepe maka ngwa ọrụ ahụ. Ngwa ọrụ enweghị ike "ịhụ" faịlụ na-anọghị n'ime folda a.

- **Ntinye**: Na-eji `chroot` (na Linux) ma ọ bụ virtual root mapping (na Windows).
- **Uru**: Na-echebe faịlụ sistemụ dị mkpa (dịka `.env`, `id_rsa`).

## L3: Nkewapụ Docker Container

"Gold Standard" maka **Ngwa Ọrụ Nwere Nnukwu Ihe Ize Ndụ** (dịka `bash`, `python`, `pip`, `curl`). Ọ bụla ọrụ ngwa ọrụ na-emepụta Docker container nwa oge site na iji onyinyo `omnipaw/tool-sandbox`.

### Gịnị kpatara L3?

- **Nkewapụ Zuru Ezu**: Ọ bụrụgodị na ngwa ọrụ enweta "root" n'ime container, ọ na-anọgide n'ime OS mebere emepe.
- **Ọnọdụ Nwa Oge**: A na-ebibi container ozugbo ngwa ọrụ rụchara ọrụ, na-ahụ na ọ nweghị faịlụ ọjọọ fọdụrụ.
- **Nchedonetwọk**: A na-ewepụ ohere netwọk site na ndabara belụsọ ma onye ọrụ nwere `PermissionGrant` doro anya.

## Ịhazi Ọkwa Sandboxing nke Ngwa Ọrụ

Ị nwere ike ịkọwa ọkwa nkewapụ n'oge ị na-edebanye aha ngwa ọrụ:

```typescript
const bashTool = new IsolatedTool({
  name: "bash",
  isolation: "L3_DOCKER", // Na-amanye Docker sandbox
});
```
