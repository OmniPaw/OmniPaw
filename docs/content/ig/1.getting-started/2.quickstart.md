# Mmalite Ngwa Ngwa

Ntuziaka a ga-enyere gị aka itinye @omnipaw/kernel ma malite usoro onye ọrụ nchekwa mbụ gị.

## Ntinye

Tinye ihe nchịkwa na ọrụ nnabata site na npm:

```bash
npm install @omnipaw/kernel
```

## Ịmalite Ihe Nchịkwa (OmniKernel)

Ịmalite ezigbo `OmniKernel` gụnyere ijikọ usoro nchekwa, itinye `ToolGate` maka ọrụ a manyere na sandbox, na ịmalite `Scheduler`.

```typescript
import {
  OmniKernel,
  KernelConfig,
  TickEngine,
  Scheduler,
} from "@omnipaw/kernel";
import {
  FileLogger,
  LlmBrainAdapter,
  ToolGate,
  ToolRegistry,
} from "@omnipaw/kernel/host";

// 1. Malite Nchekwa & Nhazi
const logger = new FileLogger("./.storage");
const config = new KernelConfig("LIVE");
const kernel = new OmniKernel(logger);

// 2. Hazie Nchebe Ngwa (Sandboxing)
const tools = new ToolRegistry();
const gate = new ToolGate(kernel.logger, config, tools.listHandlers());

// 3. Jikọọ Injinia AI & Malite OS
const scheduler = new Scheduler(
  new TickEngine(),
  kernel.logger,
  kernel.lifecycle,
  gate,
  config,
  kernel.persistentStore,
);
const brain = new LlmBrainAdapter(); // Chọrọ OPENAI_API_KEY / ANTHROPIC_API_KEY

// 4. Debanye aha ma mụọ onye ọrụ
const agent = "agent-1";
kernel.registry.registerAgent({
  id: agent,
  type: "explorer",
  createdAt: Date.now(),
});
kernel.lifecycle.transition(agent, "spawn");
kernel.lifecycle.transition(agent, "activate");

// 5. Gbaa ọsọ Tick Round
await brain.spinTickRound(
  scheduler,
  agent,
  1,
  "Investigate the system configuration.",
  {},
);
```

## Itughari Ọrụ (Replaying Execution)

Ike OmniPaw bụ ikike ọ nwere ịlaghachi azụ n'oge. Iji tugharịa oge gara aga, naanị gbanwee usoro `KernelConfig` ma rụtụ aka na faịlụ ndekọ (log file).

```typescript
// Malite na usoro REPLAY
const replayConfig = new KernelConfig("REPLAY");
const replayLogger = new FileLogger("./.storage/kernel.log");
const replayKernel = new OmniKernel(replayLogger);

// Ihe nchịkwa ga-emezi otu mgbanwe ọnọdụ ahụ dabere na ndekọ ahụ
await replayKernel.replayAll();
```
