# Nhazi Ihe Nchịkwa (Kernel Config)

Emebere OmniPaw ka ọ bụrụ ihe a pụrụ ịgbanwe agbanwe nke ọma. Nhazi ihe nchịkwa na-akọwa "Physics" nke nchịkwa gị—na-ekpebi otu iwu invariants ga-esi dị mkpa na otu hashing ga-esi gbuo oge.

## Atụmatụ Nhazi (Configuration Schema)

Ịhazi ihe nchịkwa na-eme site na klas `KernelConfig`. N'okpuru ebe a bụ blọọgụ ọchịchị ndị isi:

### 1. Ọkwa Gọvanọ (Governor Level)

`governor` na-akọwa oke nlele nke ihe nchịkwa.

| Ọkwa       | Usoro       | Nkọwa                                                                                                  |
| :--------- | :---------- | :----------------------------------------------------------------------------------------------------- |
| **LAX**    | DEVELOPMENT | A na-enyocha invariants mana mmebi na-edebanye nanị ịdọ aka ná ntị. Kachasị mma maka nnwale ngwa ngwa. |
| **STRICT** | PRODUCTION  | Ọ bụla mmebi invariant na-ebute nkwụsị ihe nchịkwa ozugbo.                                             |
| **AUDIT**  | COMPLIANCE  | Hashing trace zuru ezu na ndekọ ọnọdụ na-adịgide adịgide maka mgbanwe niile.                           |

### 2. Atụmatụ Ụjọ (Panic Strategy)

Mgbe emebiri invariant ma ọ bụ trace diverges, ihe nchịkwa na-abanye n'ọnọdụ `PANIC`. Ị nwere ike ịhazi nzaghachi:

- **HALT**: Kwụsị ọrụ onye ọrụ niile ozugbo ma kpọchie `AgentStore`. (Default)
- **ISOLATE**: Kwụsị naanị onye ọrụ mehiere ma gaa n'ihu na ndị ọzọ.
- **SNAPSHOT**: Dee nyocha ncheta zuru ezu na akwụkwọ `.storage` tupu ịkwụsị.

### 3. Oke Akụrụngwa (Resource Quotas)

Iji gbochie mmefu ego ma ọ bụ ike gwụchaa akụrụngwa, ihe nchịkwa na-amanye oke kwa onye ọrụ:

```typescript
const config = new KernelConfig({
  quotas: {
    maxTicksPerSession: 1000,
    maxToolExecutionTimeMs: 5000,
    memoryHeapLimitMb: 512,
  },
});
```

## Usoro Ọrụ (Modes of Operation)

Ihe nchịkwa nwere ike ịmuma n'usoro ọrụ dị iche iche:

### Usoro LIVE

Usoro ndabara maka ọrụ na-achị onwe ya n'ezie. A na-atụ hash ọnọdụ, mana a na-eme ka arụmọrụ dị mma.

### Usoro REPLAY

Usoro ọgụgụ naanị ebe ihe nchịkwa na-agụ site na Write-Ahead Log. Ọ na-ewepụ side-effects nke ngwa ọrụ mpụga ma weghachi nsonaazụ e debere na mbụ, na-ahụ na usoro ọrụ yiri nke mbụ.

### Usoro DEEP_TRACE

Na-enye ohere maka nlele zuru ezu maka OpenTelemetry ma na-emepụta trace waterfalls maka ntuziaka ọ bụla.
