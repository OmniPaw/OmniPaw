# LLM Adapters

**LLM Brain Adapter** bụ oghere ọgụgụ isi n'etiti model AI na ihe nchịkwa OmniPaw. Ọ bụ ya na-ahụ maka ịkọwa context ọrụ nke ihe nchịkwa gaa na prompts model nwere ike ịghọta, na ịhazi nzaghachi model ka ọ bụrụ ntuziaka a pụrụ ime eme.

## Ndị Na-enye Ọrụ A Kwadoro

OmniPaw anaghị adabere na otu model. Ị nwere ike ịgbanwe backends site n'imelite environment variables gị na nhazi `LlmBrainAdapter`.

### 1. OpenAI (Standard)

Ntụkwasị obi dị elu na oge nkwụsị dị ala maka nhazi swarm.

- **Variable**: `OPENAI_API_KEY`
- **Model Ndabara**: `gpt-4o-mini`

### 2. Anthropic (Reasoning)

Ọ kachasị mma maka nhazi ngwa ọrụ dị mgbagwoju anya na nyocha invariant.

- **Variable**: `ANTHROPIC_API_KEY`
- **Model Ndabara**: `claude-3-5-sonnet`

### 3. Local Models (Ollama)

Maka gburugburu ebe data privacy na deterministic air-gapping dị mkpa.

- **Endpoint**: `http://localhost:11434`
- **Model a tụrụ aro**: `llama3.1-70b`

## Ọmụmaatụ Ntinye (Implementation Example)

```typescript
import { LlmBrainAdapter } from "@omnipaw/kernel/host";

// Malite n'izu brain adapter nwereoke nchekwa na token
const brain = new LlmBrainAdapter({
  provider: "anthropic",
  model: "claude-3-5-sonnet-latest",
  temperature: 0, // Ọ dị mkpa maka nchekwa (determinism)
  maxTokens: 4096,
});

// Gbaa ọsọ onye ọrụ
const result = await brain.spinTickRound(
  scheduler,
  "agent-01",
  1,
  "Verify the file system integrity.",
);
```

## Ndụmọdụ Nchekwa: Temperature Zero

Maka ntugharị nchekwa 100%, anyị **na-atụ aro siri ike** ịtọ `temperature` ka ọ bụrụ `0`.

Ọ bụ ezie na LLMs na-agbanwe agbanwe, temperature efu na-ebelata drift ma na-ahụ na usoro mkpebi nke onye ọrụ na-anọgide na-agbanwe agbanwe n'ime ọtụtụ ebe kernel na-agba.

## Swarms Nwere Ọtụtụ Model

Ị nwere ike ikenye model dị iche iche na ndị ọrụ dị iche iche n'ime otu swarm iji belata ego na ịbawanye ọrụ:

- **Onye Ndu (Leader Agent)**: Jiri model nwere ọgụgụ isi dị elu (dịka GPT-4o) maka atụmatụ.
- **Ndị Ọrụ (Worker Agents)**: Jiri model dị ngwa ma dị ọnụ ala (dịka Llama 3) maka ọrụ ịkpọ ngwa ọrụ ugboro ubboro.
