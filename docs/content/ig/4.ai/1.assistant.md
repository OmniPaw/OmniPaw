# AI Assistant

Ebe nrụọrụ weebụ akwụkwọ OmniPaw nwere Onye Inyeaka AI agbakwunyere nke emebere iji nyere gị aka ịghọta mgbagwoju anya nke ihe nchịkwa nchekwa (deterministic kernel).

N'adịghị ka chatbots nkịtị, onye inyeaka a bụ **State-Aware** (ọ maara ọnọdụ). Ọ na-eji Model Context Protocol (MCP) weghachite nkọwa teknụzụ doro anya na ọmụmaatụ koodu ozugbo site na codebase na akwụkwọ OmniPaw.

## Olee otu Onye Inyeaka si arụ ọrụ?

Onye inyeaka na-arụ ọrụ dịka "Onye Ọrụ Nleta" (Guest Agent) n'ime ihe nchịkwa akwụkwọ pụrụ iche:

1.  **Ntinye (Ingestion)**: Onye inyeaka na-anata ajụjụ gị n'asụsụ nkịtị.
2.  **MWegachitere MCP**: Ọ na-akpọ mcp server nke akwụkwọ iji nweta ozi site na folda `content/` na `layer/`.
3.  **Simulation**: Maka ajụjụ gbasara koodu, onye inyeaka nwere ike iṅomi traces nke ọrụ kernel iji hụ na koodu ọ tụrụ aro na-agbaso Iwu Isi 15.
4.  **Nzaghachi**: Ọ na-achịkọta data a weghachitere ka ọ bụrụ azịza dị mkpirikpi, na-arụtụ aka na faịlụ na nọmba ahịrị doro anya.

## Atụmatụ Ndị Isi

- **Nkwado Koodu**: Na-emepụta ọmụmaatụ TypeScript nke dabara na ụdị `@omnipaw/kernel` dị ugbu a.
- **Nkọwa Trace**: Ọ bụrụ na itinye ndekọ kernel ma ọ bụ hash na-adịgide adịgide, onye inyeaka nwere ike "de-hash" usoro ahụ iji kọwaa kpọmkwem ihe onye ọrụ na-eme.
- **MCP Native**: Agbakwunyere ya nke ọma na [Model Context Protocol](/ig/ai/mcp) maka izi ezi teknụzụ nke oge a.

## Nhazi (Configuration)

A na-achịkwa onye inyeaka site na `app.config.ts`. Ị nwere ike ịgbanwe FAQ na kọntaktị mmekọrịta ya:

```ts [app.config.ts]
export default defineAppConfig({
  assistant: {
    faqQuestions: {
      ig: [
        {
          category: "Ihe Nchịkwa (Kernel)",
          items: [
            "Gịnị bụ Iwu Isi 15?",
            "Olee otu m ga-esi malite n'usoro REPLAY?",
          ],
        },
        // ... more categories
      ],
    },
  },
  socials: {
    github: "https://github.com/omnipaw/kernel",
    discord: "https://discord.gg/omnipaw",
  },
});
```

## Nchekwa & Omume ọma

Onye inyeaka bụ naanị maka ọgụgụ (read-only) ma enweghị ike ịgbanwe faịlụ project gị ma ọ bụ gbaa ngwa ọrụ na OS nnabata gị. Ọ na-agbaso otu [ToolGate](/ig/concepts/tool-permissions) governance dịka onye ọrụ OmniPaw ọkọlọtọ, na-ahụ na ọ nọgidere n'ime sandbox nke akwụkwọ.

Na-esote: Mụta banyere [Model Context Protocol (MCP)](/ig/ai/mcp).
