# Brain Adapters

**Brain Adapter** bụ usoro ntụgharị n'etiti Model Asụsụ Ukwu (LLM) na OmniKernel. Ọrụ ya bụ itunye ọdịdị probabilistic nke AI n'ime nhazi deterministic nke substrate.

## Common Adapters

OmniPaw na-enye adapters maka ndị na-enye AI isi, yana nkwado maka local inference.

| Onye Na-enye Ọrụ | Ụdị Adapter      | Ihe A Na-ejikarị Ya Ememe                           |
| :--------------- | :--------------- | :-------------------------------------------------- |
| **OpenAI**       | `OpenAiBrain`    | Nhazi swarm n'ozuzu na ọsọ.                         |
| **Anthropic**    | `AnthropicBrain` | Echiche dị mgbagwoju anya na iji ngwa ọrụ buru ibu. |
| **Ollama**       | `LocalBrain`     | Ọrụ air-gapped, 100% nke nzuzo.                     |

## Ịhazi maka Determinism

Iji hụ na onye ọrụ na-akpa agwa dịka a tọrọ aro n'oge niile, a ga-ahazi adapter nwere **Temperature 0**.

```typescript
import { LlmBrainAdapter } from "@omnipaw/kernel/host";

const brain = new LlmBrainAdapter({
  provider: "anthropic",
  model: "claude-3-5-sonnet-latest",
  temperature: 0, // Ọ DỊ MKPA: Na-egbochi logic drift n'oge replays
  maxTokens: 4000,
});
```

## Usoro Ndụ Ọrụ (Execution Lifecycle)

Mgbe brain malitere "Thought Cycle", adapter na-eme usoro ndị a:

1.  **Nchịkọta Context**: Ọ na-agụ `AgentStore` ma nyochaa data [Contextual MCP](/ig/ai/contextual-mcp) dị mkpa.
2.  **Mmepụta Ntuziaka**: Ọ na-ajụ model maka nzọụkwụ ọzọ (dịka "Kpọọ ngwa ọrụ bash ka i depụta faịlụ").
3.  **Nlele Ọchịchị**: Ihe nchịkwa na-egbochi ntuziaka emepụtara ma nyochaa ya megide [Njikwa Nchekwa](/ig/essentials/security-invariants).

## Atụmatụ Multi-Model

Na nhazi [Swarm Intelligence](/ig/ai/swarm-intelligence), ị nwere ike ikenye brain adapters dị iche iche na ndị ọrụ dị iche iche. "Onye Ndu" nwere ike iji model dị ike (Claude), ebe "Ndị Ọrụ" nwere ike iji local model dị mfe (Mistral).
