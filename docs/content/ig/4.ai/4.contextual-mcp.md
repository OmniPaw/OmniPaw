# Contextual MCP

OmniPaw na-eji **Model Context Protocol (MCP)** dịka ọkọlọtọ maka ijikọ ọnọdụ kernel na ikike brain AI. MCP na-enye ndị ọrụ AI ohere "ịjụ" gburugburu ha maka resources ma gbaa "ngwa ọrụ" n'enweghị ọpụpụ na deterministic substrate.

## Gịnị kpatara Contextual?

N'ime RAG (Retrieval-Augmented Generation) ọkọlọtọ, context na-adịkarị otu ebe. Na OmniPaw, mcp server bụ **Context-Aware** (ọ maara context). Ọ na-agbanwe resources dịịrị onye ọrụ AI dabere na `Tick` dị ugbu a na ikike doro anya nke onye ọrụ ahụ nwere.

## Atụmatụ MCP

Mcp server nke OmniPaw na-egosi ọkwa data anọ na onye ọrụ AI:

1.  **Resources (State)**: Ọnụ ahịa ozugbo site na `AgentStore`, [Ndekọ Nnyocha](/ig/essentials/audit-logs) na nso nso a, na ọnọdụ invariant.
2.  **Ngwa Ọrụ (Tools)**: Iwu dịpụrụ adịpụ nke ọma (dịka `bash`, `python`) nke [ToolGate](/ig/concepts/tool-permissions) na-achịkwa.
3.  **Prompts (Logic Guides)**: Templates echiche akọwapụtara maka ịnyagharị mmebi kernel ma ọ bụ state divergences.
4.  **Sampling (Model Feedback)**: Na-enye ihe nchịkwa ohere ịjụ onye ọrụ AI maka nzaghachi na trace ọrụ ya.

## Ntinye Teknụzụ

Ọ bụla OmniPaw na-egosi ikike MCP ya site na JSON-RPC transport nwere nchekwa. Nke a na-enye gị ohere ijikọ ọ bụghị naanị onye inyeaka agbakwunyere, kama ọ bụla MCP-compatible client.

```typescript
// Ọmụmaatụ nkọwa MCP Resource na OmniPaw
kernel.mcp.registerResource({
  uri: "omni://kernel/state",
  name: "Kernel State",
  description: "Hash ọnọdụ ozugbo na ọnọdụ nlele invariant.",
  handler: async () => ({
    hash: kernel.getStateHash(),
    invariants: kernel.getActiveInvariants(),
  }),
});
```
