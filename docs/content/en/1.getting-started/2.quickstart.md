# Quick Start

This guide will help you install @omnipaw/kernel and boot your first deterministic agent loop.

## Installation

Install the kernel and host services via npm:

```bash
npm install @omnipaw/kernel
```

## Booting a Kernel

Booting a full `OmniKernel` involves wiring the storage logic, attaching a `ToolGate` for sandboxed execution, and initializing the `Scheduler`.

```typescript
import { OmniKernel, KernelConfig, TickEngine, Scheduler } from '@omnipaw/kernel';
import { FileLogger, LlmBrainAdapter, ToolGate, ToolRegistry } from '@omnipaw/kernel/host';

// 1. Initialize Storage & Configuration
const logger = new FileLogger('./.storage');
const config = new KernelConfig('LIVE');
const kernel = new OmniKernel(logger);

// 2. Setup Tool Sandboxing
const tools = new ToolRegistry();
const gate = new ToolGate(kernel.logger, config, tools.listHandlers());

// 3. Connect AI Engine & Boot OS
const scheduler = new Scheduler(new TickEngine(), kernel.logger, kernel.lifecycle, gate, config, kernel.persistentStore);
const brain = new LlmBrainAdapter(); // Requires OPENAI_API_KEY / ANTHROPIC_API_KEY

// 4. Register and Spawn Agent
const agent = 'agent-1';
kernel.registry.registerAgent({ id: agent, type: 'explorer', createdAt: Date.now() });
kernel.lifecycle.transition(agent, 'spawn');
kernel.lifecycle.transition(agent, 'activate');

// 5. Execute a Tick Round
await brain.spinTickRound(scheduler, agent, 1, "Investigate the system configuration.", {});
```

## Replaying Execution

The power of OmniPaw is its ability to travel back in time. To replay a previous session, simply switch the `KernelConfig` mode and point to the log file.

```typescript
// Boot in REPLAY mode
const replayConfig = new KernelConfig('REPLAY');
const replayLogger = new FileLogger('./.storage/kernel.log');
const replayKernel = new OmniKernel(replayLogger);

// The kernel will now execute identical state transitions based on the log
await replayKernel.replayAll();
```
